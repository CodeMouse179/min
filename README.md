# min

- min text editor

## intro

min is a modern cross-platform C11 text editor.

## build & run

Windows: cmake . && min_project.sln && run(Ctrl + F5)

POSIX: cmake . && make && ./min

## PATH

Windows: Google it :)

macOS: cd /etc/paths.d && sudo vim min_project, "input path of min", save: (esc+:+w+q+enter)
